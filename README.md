# virtualhelper.site

**Python 3.X**


**Copiando variavel de exemplo para principal**
```
cp .env.example .env
```

**Instalando dependencias**
```
pip install -r requirements.txt
```

**Rodando servidor**
```
flask run
```

## Windows

```
$env:FLASK_APP="src.base.autoload"
```

```
$env:FLASK_ENV="development"
```

## MYSQL CONFIGURACAO

Esta tudo dentro do .env
